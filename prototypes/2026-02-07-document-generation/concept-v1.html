Prototype Notes

Screen 1: Settings Templates tab — simple list rows matching existing settings patterns. "Use" button appears on hover (pill-shaped, surface-inverse). No search/filter/grid — settings is for quick access, not browsing.

Screen 2a: Generation modal — two-column layout. Left: company selector + auto-filled variables with green checkmarks + manual fields. Right: section preview showing which are AI-generated. Footer: Cancel + Generate Document buttons.

Screen 2b: Generation progress — progress bar + section checklist. Complete = filled circle with check. Generating = spinner. Pending = empty circle. Clean, no noise.

Screen 3: Document editor — full-page view with back arrow + title in header. Export + Save in header. Sections as content blocks with numbered labels. AI sections show "Regenerate" on hover. Content areas have subtle border on focus for editability cue.

Design decisions: All buttons pill-shaped. Surface-inverse for primary actions. Hover-reveal for secondary actions (Use, Regenerate). No AI gradients or sparkles (AI-assumed design). Section numbers as muted tabular figures. Consistent focus rings.
