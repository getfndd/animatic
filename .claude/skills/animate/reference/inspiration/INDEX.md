## Pattern Categories for Primitives Extraction

The references above cluster into these extractable patterns:

1. **Grid ripple / wave** — Dot or element grids with traveling distortion (3 refs)
2. **Stagger cascade** — Sequential reveal across a grid or list (4 refs)
3. **Kinetic typography** — Text as the animated subject (4 refs)
4. **3D perspective motion** — Cards, blocks, or elements with Z-axis rotation (3 refs)
5. **Ambient field** — Continuous background motion for texture (3 refs)
6. **Materialization** — Elements assembling from particles or fragments (1 ref)
7. **Parametric curves** — Mathematical/generative motion (2 refs)
