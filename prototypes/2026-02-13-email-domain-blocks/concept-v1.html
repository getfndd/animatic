Implementation Notes

MetricCard Extension

  • Attrs: label, value, change, trend
  • Trend: up | down | neutral
  • HTML: div with data-metric-card
  • Email: table cell with inline styles
  • Group: Auto-rows when 2+ adjacent
  • Colors: Green for up, red for down, gray for neutral

Columns Extension

  • Attrs: columnCount (2 or 3)
  • Content: Each column is a nested text editor
  • HTML: div with data-columns
  • Email: table-based layout (no flexbox)
  • Mobile: Stacks vertically at narrow widths
  • Note: NOT atom — contains editable content

Ask/Request Extension

  • Attrs: heading, body, ctaLabel, ctaUrl, variant
  • Variant: ask | request (differentiated by label, not border)
  • HTML: div with data-ask-block
  • Email: card surface with rounded corners, no callout borders
  • CTA: Optional — same pattern as CtaButton
  • Dark: Inverted surface + button colors
