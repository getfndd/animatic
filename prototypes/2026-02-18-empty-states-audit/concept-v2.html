Design Decisions

Ghost Previews

Used on Investors, Pipeline, Data Room, and Deal Docs — pages where showing the populated layout helps users understand what they're building toward. Inspired by Figma's approach. Ghost elements use opacity 35% and are non-interactive.

Copy Tone

Shifted from clinical ("There are no records to display") to encouraging ("Build your investor list"). Added context about why ("Most successful seed rounds start with 40-60 targets") without being preachy. Body text explains value, not just mechanics.

Dual CTAs

Allowed on Investors, Data Room, Contacts where there's a clear primary + secondary path (Add vs Import, Template vs Manual). Secondary path uses tertiary-action style (no border, text-secondary). This is a spec update — the single-CTA rule was too restrictive for onboarding contexts.

Spec Alignment

All empty states now use: text-lg font-semibold heading, text-sm text-tertiary body, text-tertiary icon color, 64px/32px icon sizing, secondary-action buttons, mt-6 button spacing (updated from mt-4 — the extra breathing room is better).
