When there are zero signals, showing a big "0" with "Cold" badge and three zero counters (Emails 0, Meetings 0, Documents 0) is visual noise that tells the user what they already know — nothing happened. It also makes "Cold" feel like a judgment rather than a data point.

Interaction card icon containers use rounded-full (deferred)

The interaction cards in Notes & Meetings use rounded-full for icon containers. Per design system, icon containers should be rounded-md. This exists in the current code and is out of scope for FND-1423, but worth noting.
