# Ogilvy Learning Governance (REFLEX)

## Purpose

Controls how Ogilvy's marketing and positioning knowledge evolves. Prevents drift from core voice principles while allowing refinement based on real market feedback.

## Categories

### Positioning
- How we frame Preset's value proposition
- Audience-specific messaging angles
- Competitive differentiation claims

### Messaging
- Headline patterns that resonate
- Copy structures that convert
- Register-appropriate phrasing

### Competitive Intel
- Competitor positioning shifts
- Market category evolution
- Differentiation opportunities

### Conversion
- CTA patterns that work
- Funnel stage messaging
- Proof point effectiveness

## Learning Rules

### What Gets Captured

| Trigger | Action | Category |
|---------|--------|----------|
| User confirms a positioning angle works | Record angle + context | Positioning |
| Headline/copy is approved for use | Record pattern + register | Messaging |
| Competitor changes positioning | Record shift + our response | Competitive Intel |
| Conversion data shared | Record pattern + metric | Conversion |

### What Does NOT Get Captured

- One-off copy requests without feedback
- Speculative positioning not validated
- Personal preferences without rationale
- Competitor gossip without strategic implication

## Validation Requirements

### Before Recording a Learning

1. **Specificity** — Is the learning specific enough to act on?
2. **Evidence** — Is there user confirmation, data, or clear rationale?
3. **Repeatability** — Would this apply beyond the single instance?
4. **Consistency** — Does it align with voice guide principles?

### Before Applying a Learning

1. Check register context (landing page vs tweet vs blog)
2. Verify proof points are still current
3. Confirm audience alignment
4. Ensure no conflict with existing learnings

## Conflict Resolution

When learnings conflict:

1. More recent market data wins over older assumptions
2. User-validated > team-preferred
3. Data-backed > intuition-based
4. Specificity wins (audience-specific > generic)

## Anti-Drift Rules

These principles are FIXED and learnings cannot override them:

- Never use superlatives without proof
- Never use emojis in marketing copy
- Proof over claims, always
- Buyer-centric framing, never product-centric
- Anti-hype stance is non-negotiable

## Review Cadence

- After each positioning session: capture what resonated
- After competitive research: update landscape understanding
- After user feedback: refine messaging patterns
- Monthly: review all learnings for consistency
